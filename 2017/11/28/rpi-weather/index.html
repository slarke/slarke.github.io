Raspberry pi 2 Погода

2017-11-28
#manual #DIY

https://www.adafruit.com/product/1601

https://learn.adafruit.com/adafruit-pitft-28-inch-resistive-touchscreen-display-raspberry-pi
Готовый скрипт Adafruit работает Debian 8 (jessie), но новых версиях не работает. Последний образ raspbian jessie
https://downloads.raspberrypi.org/raspbian/images/raspbian-2017-07-05/
После установки образа raspbian, необходимо закинуть пустой файл ssh, тогда будет возможность подключаться.

Установка

sudo adafruit-pitft-helper -t 28r

MagicMirror популярен, но версии Raspberry Pi B+ и более ранние поддерживают только режим сервера. Поэтому для отображения выбрал PiClock

подробная установка https://github.com/n0bel/PiClock/blob/master/Documentation/Install-Clock-Only.md

Потребуется api key https://www.wunderground.com/weather/api/
